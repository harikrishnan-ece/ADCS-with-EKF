# ADCS with EKF Simulation

This project demonstrates a simulation of a Satellite Attitude Determination and Control System (ADCS) using an Extended Kalman Filter (EKF).

## Project Overview
The aim of this project is to simulate how a satellite estimates and controls its orientation in space. The Extended Kalman Filter is used to estimate the satellite attitude using sensor measurements.
